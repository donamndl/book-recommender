# 📚 Book Recommendation System

An end-to-end Machine Learning based Book Recommendation Web Application built using Flask and collaborative filtering.

This system recommends books to users based on similarity scores computed using a trained ML model.

---

## 🚀 Features

- Popular books display
- Similar book recommendations
- Collaborative filtering based recommendation
- Clean and responsive UI using HTML templates
- Pre-trained similarity model

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- Pickle
- HTML/CSS

---

## 💡 How It Works

- User selects a book.
- System finds similar books using cosine similarity.
- Top recommended books are displayed with details.
